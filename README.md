# FG - HSS website #

## Introduction ##
For this team assignment we have cooperated with a real client - HSS (Hässelby Strands sjöscoutkår) and built their website.

## Live Demo ##
Check out HSS [website](https://hss-website-two.vercel.app/).

## Tech Stack ##
- HTML5
- CSS
- Vanilla Javascript
- Tailwind
- Nextjs
- Typescript
- deployed on Vercel
